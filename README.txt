VarAC Weekly results updates

The following python scripts were written to update the historic check-in results

varAC-Parse.py - Script that parses input results file and creates a CSV listing date and callsign of check-in
varAC-totals.phy - Script that tallies number of times a given callsign logged in across all dates
varAC-totals-for-web.py - Script that formats output of previous script for publishing on the web


The results.txt file in check-in-results is updated each week to with new results.

This file is parses each week to produce an updated listing for posting to the web.
